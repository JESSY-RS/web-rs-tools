【J式計算方法】
所要時間 / {(戦闘時間 / 成長判定回数) + 間隔時間} = 可能回数
可能回数 / 100(100戦ごとに入れ替え想定) * 育成枠数 = 可能人数
可能人数 * (成長確率 / 100) * ボーナス倍率 = 期待値

【M式計算方法】まこさん @mima_ma_ 感謝
(戦闘時間 + 間隔時間) / 育成枠数 / 成長判定回数 / ボーナス倍率 = 期待値
