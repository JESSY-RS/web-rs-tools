プライバシーポリシー

本ポリシーは、JESSY が提供するブラウザ拡張機能（以下「本拡張機能」）における利用者情報の取り扱いを説明するものです。

1. 個人情報の収集について

提供する全ての拡張機能（ScrollToTopJ, XEmojiHistorySaver 等）において、利用者の個人情報、閲覧履歴、入力内容、Cookie情報などのデータを収集・蓄積・外部サーバーへ送信することは一切ありません。

2. 権限の使用目的について

本拡張機能が要求する権限は、以下の目的のためにのみ使用されます：

  • ScrollToTopJ: ページのトップへスクロールする操作、および設定（アイコン色）の保存。
  • XEmojiHistorySaver: X(Twitter)上での絵文字入力の検知、履歴のローカル保存、およびクリップボードへのコピー。

これらのデータは全て利用者のブラウザ内（ローカルストレージ）でのみ管理され、外部に漏洩することはありません。

3. 外部サービスの利用について

本拡張機能は、分析や広告などの目的で第三者のサービスを利用することはありません。

4. ポリシーの変更について

法令の制定、改正等により、本ポリシーを適宜見直し、予告なく変更する場合があります。

5. お問い合わせ

本ポリシーに関するお問い合わせは、各ストア（Chrome ウェブストア / Edge アドオン）のデベロッパー連絡先、または GitHub の Issue にてお願いいたします。
