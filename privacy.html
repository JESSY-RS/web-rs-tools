プライバシーポリシー

本ポリシーは、JESSY が提供するブラウザ拡張機能、および Webアプリケーション（ドライブモードを含む各種ツール群）における利用者情報の取り扱いを説明するものです。

第1部：ブラウザ拡張機能について

1. 個人情報の収集について

提供する全ての拡張機能（ScrollToTopJ, XEmojiHistorySaver 等）において、利用者の個人情報、閲覧履歴、入力内容、Cookie情報などのデータを収集・蓄積・外部サーバーへ送信することは一切ありません。

2. 権限の使用目的について

本拡張機能が要求する権限は、以下の目的のためにのみ使用されます：

  • ScrollToTopJ: ページのトップへスクロールする操作、および設定（アイコン色）の保存。
  • XEmojiHistorySaver: X(旧Twitter)上での絵文字入力の検知、履歴のローカル保存、およびクリップボードへのコピー。

これらのデータは全て利用者のブラウザ内（ローカルストレージ）でのみ管理され、外部に漏洩することはありません。

第2部：Webアプリケーション（ドライブモード等）について

各種チェッカーツール等で提供される「ドライブモード（クラウド同期機能）」をご利用の際における情報の取り扱いは以下の通りです。

1. Googleアカウント情報の利用について

ドライブモードでは、データの同期を行うために Google Identity Services を用いた認証を行います。認証は Google のセキュアな標準システムを通じて行われ、アプリ側で利用者のパスワードや個人を特定する情報を取得・保持することはありません。

2. セーブデータの保存先について

ドライブモードで保存されるセーブデータ（チェッカーの入力状況など）は、利用者ご自身の Google Drive 内の「アプリ専用隠しフォルダ（App Data フォルダ）」にのみ保存されます。開発者（JESSY）および第三者がこのデータにアクセスしたり、閲覧・取得したりすることは技術的に不可能です。

3. サーバーへのデータ送信について

当サイトの Web サーバー（Firebase Hosting 等）に対し、利用者のセーブデータや個人情報を無断で送信・蓄積することはありません。同期処理は、利用者のブラウザと Google Drive 間で直接通信が行われます。

4. 例外事項：開発者用テストモードについて

本ツールには、開発およびテストを目的とした特殊な動作モード（隠しパラメータを用いたアクセス）が存在します。このモードを利用してアクセスした場合に限り、本プライバシーポリシーの「データの保存先」に関する規定は適用されず、データは利用者自身の Google Drive ではなく、外部のクラウドサーバー（Firebase）に保存されます。
このモードは一般利用を想定したものではないため、利用者が意図的またはソースコードの解析等によりアクセスした場合であっても、個人情報の完全な保護は保証されません。一般の利用者は、必ず通常のURLからアクセスし、通常モード（Google Drive連携）をご利用ください。

第3部：共通事項

1. 外部サービスの利用について

本拡張機能および Webアプリケーションは、アプリの機能提供（Google Drive API等）やホスティングの目的で外部サービスを利用しますが、利用者の行動追跡や広告を目的とした第三者のトラッキングサービス（アナリティクス等）は使用していません。

2. ポリシーの変更について

法令の制定、改正等により、本ポリシーを適宜見直し、予告なく変更する場合があります。重要な変更がある場合は、サービス内にてお知らせいたします。

3. お問い合わせ

本ポリシーに関するお問い合わせは、各ストア（Chrome ウェブストア / Edge アドオン）のデベロッパー連絡先、または GitHub の Issue にてお願いいたします。
